A Streamlit-based machine learning web app for detecting fraudulent transactions.
This project demonstrates data preprocessing, model training (using Logistic Regression / XGBoost), and deployment inside a Docker container.

**Features**

- Interactive Streamlit UI
- Fraud prediction using trained ML model
- Clean and modular Python pipeline
- Fully containerized using Docker
- Deployable on Render, AWS, Azure, or GCP

**Tech Stack**

- Python 3.11
- Streamlit
- Pandas, NumPy, Scikit-learn, XGBoost
- Matplotlib, Seaborn
- Docker
